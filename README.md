## 卡片生成器

在之前公司做的一个用来合成图片的工具。使用canvas实现图片合成，裁图使用人家开源的裁图插件[PhotoClip.js](https://github.com/baijunjie/PhotoClip.js)。

通过本项目，你可以初步了解如何使用webpack打包压缩代码，如何使用canvas来进行图片文字合成。

[在线demo](https://auven.github.io/cards-maker/dist/)
